Scratchnux

echo - Write out a phrase in the console.
ls - See all the files.
download [file] - Download a file to the host computer.
upload [filename] - Upload a file from the host computer to the emulator with a specific filename.
reboot - Restart the machine.
exit - Shutdown the machine.
bios [krnl] [options] - BIOS Config. (don't run bios krnl shutoff)
iframe [url] - Open up an iframe(click the gray rectangle at the top right to exit)
prompt - Ask a prompt then output it in $VAR.
var [variable] [text] - Set a variable to text.
ifeq [variable] [equal to] [then] [else] - Write out [then] if a variable is equal to [equal to], else write out.
sleep [seconds] - Wait [seconds] seconds.
bash - Restart the terminal.
style - Style the terminal
unstyle - Remove the style.
view [file] - View a file.
exec [file] - Execute a command. (COMING SOON.)
web [url] [filename] - Download a file from the internet.
root [true/false] - Give yourself superuser permissions.
gui [load/setbg] [options] - Load the GUI. (DEPRECATED.)
hostname [set] [username] - Set your hostname.
delete [--no-file-refresh] [file] - Delete a file.
filerefresh - Refresh the files in the VM.
prjstrg [restore/connect/backup] - Connect to a server and restore your data.
